FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS, and JavaScript. It´s suitable for all levels and ideal for portfolio building.

Is Frontend Mentor free?

Yes, Frontend Mentor is free.

Can I use Frontend Mentor projects in my portfolio?

Of course yes, when solving Frontend Mentor challenges, You can select the best challenges and display them in your Portfolio. Helping developers add professional-looking projects to their portfolio was one of the reasons the platform was created!

How can I get help if i´m stuck a challenge?

The best (and fastest) way to get help with a challenge is on our Discord server. There are thousands of other developers there, so it's a great place to ask questions. We even have a dedicated “help” channel! If you haven't signed up yet, you can receive an invite to our Discord server here.
